# IBM Data Analyst Professional Certificate — Capstone Project

## 📌 Project Overview
This capstone project is the final component of the IBM Data Analyst Professional Certificate on Coursera.
It demonstrates the end-to-end data analysis skills acquired throughout the program, including data collection,
wrangling, exploratory analysis, visualization, and storytelling with data.

## 🛠️ Tools & Technologies
- **Python** (Pandas, NumPy, Matplotlib, Seaborn)
- **SQL**(mysql,sqlite)
- **Tableau**
- **Jupyter Notebook**
- **Excel**


## 📊 Key Tasks
1. **Data Collection** — Web scraping & API calls to gather survey data
2. **Data Wrangling** — Cleaning, handling missing values, and normalization
3. **Exploratory Data Analysis (EDA)** — Statistical analysis and trend identification
4. **Data Visualization** — Charts and plots to communicate insights
5. **Dashboard Creation** — Interactive dashboards using IBM Cognos
6. **Storytelling** — Final presentation summarizing findings

## 📈 Dataset
The project uses the **Stack Overflow Developer Survey** dataset, which captures
insights on programming languages, tools, and technologies used by developers worldwide.


## 📜 Certificate
This project is part of the [IBM Data Analyst Professional Certificate](https://www.coursera.org/professional-certificates/ibm-data-analyst) on Coursera.
